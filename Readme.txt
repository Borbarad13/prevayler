Requirements
================
Prevayler requires Java 1.2 or greater.
The GUI for Prevayler's demo application requires Java 1.3 or greater.

This version of Prevayler was tested using JDK1.3.1_01.


Build Procedure
===================
To build and run Prevayler you compile the source code directly. This ensures maximum JDK compatibility.


Demo Application
====================
The prevayler.demo.PrevaylerDemo class is a bank application that uses Prevayler. It has the main() method and can be run directly. It is self explanatory.

Requires Java 1.3 or greater because of the GUI classes used.
